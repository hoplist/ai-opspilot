# OpsPilot Deploy

Deployment manifests for the clean-slate OpsPilot architecture.

Default modules:

- `core/`
- `mcp/`
- `console/`
- `worker/`
- `rbac/`

Optional modules:

- `optional/opensearch/`
- `optional/minio/`
- `optional/mysql/`
- `optional/ebpf/`

Default deployment should not include optional modules.
