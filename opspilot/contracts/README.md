# OpsPilot Contracts

Contracts are defined before implementation.

Planned contract files:

- `openapi.yaml`
  - Backend REST API.
- `cli-schema.json`
  - CLI command schema for humans and AI skills.
- `mcp-tools.json`
  - MCP tool names, parameters, read-only boundary, and response shape.
- `evidence-pack.schema.json`
  - Evidence Pack JSON structure.
